Help & Information

📋 Barcode Formats

  • Code 128: Most common, handles all ASCII characters, good for inventory
  • Code 39: Alphanumeric, simple to use
  • EAN-13: European standard, 13 digits required
  • UPC-A: US standard, 12 digits required

📱 How to Scan

  1. Go to "Scan Barcode" tab
  2. Click "Start Scanner"
  3. Allow camera access when prompted
  4. Point camera at barcode
  5. Wait for automatic detection
  6. Barcode number will appear below

🔧 How to Generate

  1. Go to "Generate Barcode" tab
  2. Enter barcode number (numbers only)
  3. Select format from dropdown
  4. Click "Generate"
  5. Preview will appear below
  6. Use "Download PNG" to save as image
  7. Use "Copy" to copy to clipboard

📦 Batch Generation

  1. Go to "Batch Generate" tab
  2. Paste multiple barcode numbers (one per line)
  3. Choose format and height
  4. Click "Generate All"
  5. All barcodes preview below
  6. Download as ZIP or Print directly

💡 Tips & Tricks

  • Use 100-150px height for printing
  • Code 128 is most compatible with scanners
  • For best scanning, use good lighting
  • EAN-13 requires exactly 13 digits
  • Keep barcodes away from edges when printing
  • Test scan with scanner hardware before using in production

🔗 Integration with Cycle Count App

Use this tool to generate test barcodes for the Cycle Count application. Generate barcodes matching your inventory (11011, 11012, 11013, etc.), print them, and use them to test the scanner functionality.

🆘 Troubleshooting

  • Scanner not working: Check browser permissions for camera
  • Barcode not scanning: Ensure barcode is in focus, try different angle
  • Download not working: Check browser's download permissions
  • Format error: Check barcode follows format requirements (e.g., EAN-13 needs 13 digits)
